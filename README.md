# Blaze
Grocery delivery frontend made in React

## Getting Started
Clone respository into your project folder

git clone `https://github.com/joshipiyush9969/Blaze.git`

Run 
```
cd Blaze
npm install
npm run dev
```
<br/>
⚠️ react-lottie will give warning as it React 18 doesn't support it
